财经日历
接口：eco_cal
描述：获取全球财经日历、包括经济事件数据更新
限量：单次最大获取100行数据
积分：2000积分可调取
输入参数
名称
类型
必选

str
N
日期（YYYYMMDD格式）

str
N
结束日期

str
N
货币代码
country
str
N
国家（比如：中国、美国）
event
str
N
事件
（支持模糊匹配：
*非农*）

名称
类型

接口示例

pro
=
ts.pro_api()


#获取指定日期全球经济日历
df
=
pro.eco_cal(date='20200403')


#获取中国经济事件
df
=
pro.eco_cal(country='中国')

#获取美国非农数据
df
=
pro.eco_cal(event='美国季调后非农*',
fields='date,time,country,event,value,pre_value,fore_value')
